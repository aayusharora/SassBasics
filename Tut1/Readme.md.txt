To install Sass
1. Install RubyGems for Windows .
2. Open Command Prompt with Ruby.
3. Run the command  gem install sass.
4. Make a folder and save a file say style.scss.
4. Run the command sass --watch style.scss:style.css .
5. Now the sass will have a watch on the changes you will make in your style.scss.
6. In style.scss type for your first Sass program .
